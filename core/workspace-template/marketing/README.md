# marketing/

Home for the Marketing role agent's artifacts: campaigns, content, SEO
audits, brand notes, and performance metrics. See
`core/skills/marketing/SKILL.md` for the full charter.

Suggested contents:
- `campaigns/<campaign-name>.md` — brief, plan, status, results
- `content-calendar.csv` — planned/published content with dates and channels
- `brand-notes.md` — voice/tone reference
- `seo-audit-<date>.md` — point-in-time SEO findings
- `campaign-metrics.csv` — performance numbers

Other role agents may read this folder for context (e.g. Support checking
current campaigns before responding to a customer). Only the Marketing
agent should write here.
