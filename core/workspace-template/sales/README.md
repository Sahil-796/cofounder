# sales/

Home for sales pipeline, prospect notes, and deal tracking. No dedicated
"Sales" role agent ships in v1 (see `cofounder-ideation.md` §6 — launch
roles are Marketing and Research, with Support/Operations/Finance also
implemented per PLAN.md Task B); until a Sales role agent exists, the
orchestrator or Research agent may use this folder directly when the
founder asks for prospecting/pipeline work.

Suggested contents:
- `pipeline.csv` — company, contact, stage, value, next step, last touched
- `prospects/<name>.md` — notes on a specific prospect/deal
- `deals-closed.csv` — historical record of closed-won/closed-lost deals
