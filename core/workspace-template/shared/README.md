# shared/

Cross-role artifacts every role agent should read before producing
customer-facing or brand-facing work. This folder is the company's
single source of truth for things every role needs to agree on.

Suggested contents:
- `brand-voice.md` — tone, voice, messaging guardrails
- `icp.md` — ideal customer profile / target audience
- `company-goals.md` — current quarter/year goals and priorities
- `product-faq.md` — canonical product facts (used by Support, referenced
  by Marketing)

Any role agent may read this folder. Only write here when a change is
genuinely company-wide (not role-specific) — role-specific notes belong in
that role's own folder.
