# research/

Dedicated Research role agent workspace: competitor analysis, market
research, and company/people intel. See `core/skills/research/SKILL.md`
for the full charter.

This is the one folder every other role agent is expected to check before
doing its own research, to avoid duplicate work.

Suggested contents:
- `competitors/<name>.md` — profile, product, pricing, positioning
- `market/<topic>.md` — market sizing/trend notes with sources
- `intel/<company-or-person>.md` — point-in-time intel snapshots
- `watchlist.csv` — what's being tracked recurrently, cadence, last checked

Every finding must cite sources (URL + access date) — unsourced research
isn't usable by other roles.
